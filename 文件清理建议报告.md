## 🚀 执行清理

### 方式1: 自动清理（推荐）

已生成自动清理脚本：

```powershell
.\执行清理.ps1
```

这个脚本会自动删除：
- 旧版本文档 (V7.x, V8.0-V8.8)
- 测试脚本
- 临时文件
- 备份文件

### 方式2: 手动清理

如果你想更谨慎，可以手动删除：

```powershell
# 1. 删除旧版本文档
Remove-Item "V7.*.md"
Remove-Item "V8.[0-8]*.md"

# 2. 删除测试脚本
Remove-Item "测试*.py"
Remove-Item "验证*.py"

# 3. 删除诊断工具
Remove-Item "诊断*.py"
Remove-Item "诊断*.ps1"

# 4. 删除A/B电脑脚本
Remove-Item "A电脑*.ps1"
Remove-Item "B电脑*.ps1"
Remove-Item "AB电脑*.md"

# 5. 删除临时文件
Remove-Item "debug_output.txt"
```

### 方式3: 归档历史文档

```powershell
# 创建归档目录
New-Item -ItemType Directory -Path "docs/archive" -Force

# 移动历史文档
Move-Item "*分析报告.md" "docs/archive/"
```

## ⚠️ 注意事项

1. **备份重要数据**: 清理前确保数据库和 `实际数据/` 目录已备份
2. **Git提交**: 清理前先提交当前代码到Git
3. **测试运行**: 清理后运行 `.\启动看板.ps1` 确保程序正常
4. **虚拟环境**: 不要删除 `.venv/` 目录
5. **数据文件**: 不要删除 `实际数据/` 和 `学习数据仓库/` 目录
