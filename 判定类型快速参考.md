# 判定类型快速参考

## 💰涨价(销量增)

### 定义
```
单价变化率 > 5% 且 销量变化 ≥ 0
```

### 示例
```
商品: 网红零食
之前: ¥8, 50件
当前: ¥10, 60件
→ 涨价25%, 销量+10件
→ 💰涨价(销量增)
```

### 含义
- ✅ **涨价策略成功**
- ✅ **品牌力强，价格不敏感**
- ✅ **表现良好**

### 为什么出现在"下滑分析"中？
- ❓ 命名不当！
- 可能在TOP排名中下滑（被其他商品超越）
- 但商品本身表现很好

---

## ⚪已下架

### 定义
```
当前库存 = 0 且 之前库存 = 0
```

### 示例
```
商品: 季节性商品
9月28日: 库存0, 销量0
9月29日: 库存0, 销量0
9月30日: 库存0, 销量0
→ ⚪已下架
```

### 含义
- ℹ️ **商品已下架**（或从未上架）
- 📝 **无销售记录**
- ❌ **无需处理**

### 与🔴售罄的区别
```
🔴售罄:
  之前库存 > 0, 当前库存 = 0
  → 之前有货，卖光了

⚪已下架:
  之前库存 = 0, 当前库存 = 0
  → 一直没货，已下架
```

## 🎯 完整判定类型对照表

### 需要关注（问题类）

| 图标 | 名称 | 条件 | 含义 | 行动 |
|------|------|------|------|------|
| 🔴 | 售罄 | 库存从有到无 | 卖光了 | 紧急补货 |
| ⚠️ | 滞销预警 | 有库存但无销量 | 没人买 | 促销或下架 |
| ⚠️ | 库存不足 | 库存<5且销量降 | 即将售罄 | 及时补货 |
| 💰 | 涨价导致销量降 | 涨价>5%且降量 | 涨价失败 | 考虑降价 |
| 💸 | 降价仍降量 | 降价>5%且降量 | 降价无效 | 深度诊断 |
| 📉 | 销量大幅下滑 | 销量降>30% | 销量暴跌 | 紧急分析 |
| 📉 | 销量小幅下滑 | 销量微降 | 销量下降 | 持续观察 |

### 表现良好（状态类）

| 图标 | 名称 | 条件 | 含义 | 行动 |
|------|------|------|------|------|
| 💰 | **涨价(销量增)** | 涨价>5%且销量不降 | **涨价成功** | **继续保持** |
| 💸 | 降价促销成功 | 降价>5%且销量增 | 降价有效 | 保持策略 |
| ✅ | 正常 | 无明显变化 | 一切正常 | 继续保持 |

### 状态说明（记录类）

| 图标 | 名称 | 条件 | 含义 | 行动 |
|------|------|------|------|------|
| ⚪ | **已下架** | 库存一直为0 | **已下架** | **无需处理** |

## 🔑 关键理解

### 命名问题
```
字段名: "下滑原因"
问题: 不是所有情况都是"下滑"

更准确的名称:
  ✅ "业绩变化原因"
  ✅ "状态说明"
  ✅ "商品诊断"
```

### 分类建议
```
【需要关注】 - 红色/黄色
  🔴售罄
  ⚠️滞销预警
  💰涨价导致销量降
  📉销量下滑

【表现良好】 - 绿色
  💰涨价(销量增) ← 这个！
  💸降价促销成功
  ✅正常

【状态记录】 - 灰色
  ⚪已下架 ← 这个！
```

---

**快速记忆**:
- 💰涨价(销量增) = **好事！** 涨价还能增量
- ⚪已下架 = **中性** 只是状态说明
