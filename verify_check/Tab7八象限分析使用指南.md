## 🎯 功能概述

八象限分析通过**营销占比 × 毛利率 × 动销率**三个维度,将所有商品分为8个象限,帮助您快速识别:
- ⭐ **黄金商品**: 低营销+高毛利+高动销 → 最理想状态
- ❌ **双输商品**: 高营销+低毛利+低动销 → 立即止损
- 💎 **潜力商品**: 低营销+高毛利+低动销 → 值得投资
- 🔴 **引流亏损**: 高营销+低毛利+高动销 → 需要优化

## 🔧 功能使用

### 1️⃣ 象限筛选

**下拉菜单位置**: 页面左上方

**可选象限**:
- **全部象限** - 显示所有3,889个商品的统计
- **⭐ Q5-黄金商品** - 1,505个 (低营销+高毛利+高动销)
- **💰 Q1-金牛过度** - 696个 (高营销+高毛利+高动销)
- **💎 Q6-潜力商品** - 892个 (低营销+高毛利+低动销)
- **🎯 Q7-引流爆款** - 175个 (低营销+低毛利+高动销)
- **🔴 Q3-引流亏损** - 140个 (高营销+低毛利+高动销)
- **⚠️ Q2-高成本蓄客** - 251个 (高营销+高毛利+低动销)
- **❌ Q4-双输商品** - 81个 (高营销+低毛利+低动销)
- **🗑️ Q8-淘汰区** - 149个 (低营销+低毛利+低动销)

**操作**:
1. 点击下拉菜单
2. 选择感兴趣的象限
3. 页面自动更新统计卡片和商品列表

---

### 2️⃣ 统计卡片

**显示内容** (根据筛选象限动态变化):
- 商品数量
- 营销总成本
- 营销占比

**颜色标识**:
- 🔴 红色边框: P0级 (双输商品)
- 🟡 黄色边框: P1/P2级 (需要优化)
- 🟢 绿色边框: OK级 (健康商品)

**示例**:
```
⭐黄金商品
商品数: 1505
营销成本: ¥25,473
营销占比: 36.0%
```

### 4️⃣ 导出功能

**按钮位置**: 页面右上方 **"📥 导出分析结果"**

**导出字段** (Excel格式):
1. 商品名称
2. 象限编号 (Q1~Q8)
3. 象限名称
4. 优先级 (P0~P4/OK)
5. 优化建议
6. 营销等级 (高营销/低营销)
7. 毛利等级 (高毛利/低毛利)
8. 动销等级 (高动销/低动销)
9. **商品原价** ✓
10. **实收价格** ✓
11. **利润额** ✓
12. **毛利率** ✓
13. **月售** (销售量) ✓
14. **剩余库存** ✓
15. 营销总成本
16. 营销占比
17. 营销ROI

**文件命名**: `八象限分析_{象限编号}_{时间戳}.xlsx`

**示例**:
- 选择"全部象限" → 导出全部3,889个商品
- 选择"Q4-双输商品" → 仅导出81个双输商品
- 文件名: `八象限分析_ALL_20251114_153020.xlsx`

**操作步骤**:
1. (可选) 选择要导出的象限
2. 点击 **"📥 导出分析结果"** 按钮
3. 浏览器自动下载Excel文件

## 📊 数据解读

### 阈值定义

| 维度 | 高 | 低 |
|------|----|----|
| 营销占比 | ≥50% | <50% |
| 毛利率 | ≥30% | <30% |
| 动销率 | ≥中位数 | <中位数 |

### 优先级说明

- **P0**: 立即处理 (双输商品)
- **P1**: 紧急优化 (金牛过度+引流亏损)
- **P2**: 战术调整 (高成本蓄客)
- **P3**: 机会探索 (潜力商品)
- **P4**: 考虑淘汰 (淘汰区)
- **OK**: 保持监控 (黄金商品+引流爆款)

### 商品数量分布

```
OK级 (健康): 1,680个 (43.2%)
  ├─ Q5黄金商品: 1,505个
  └─ Q7引流爆款: 175个

P1级 (紧急): 836个 (21.5%)
  ├─ Q1金牛过度: 696个
  └─ Q3引流亏损: 140个

P3级 (机会): 892个 (22.9%)
  └─ Q6潜力商品: 892个

其他: 481个 (12.4%)
  ├─ Q2高成本蓄客: 251个 (P2)
  ├─ Q8淘汰区: 149个 (P4)
  └─ Q4双输商品: 81个 (P0)
```
