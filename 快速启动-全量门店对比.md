# 快速启动 - 全量门店对比功能

## 🚀 启动步骤

### 1. 启动后端服务

```bash
cd backend
python -m app.main
```

后端将在 `http://localhost:8000` 启动

### 2. 启动前端服务

```bash
cd frontend-react
npm run dev
```

前端将在 `http://localhost:5173` 启动

### 3. 访问页面

打开浏览器访问：`http://localhost:5173`

点击左侧导航栏的 **"全量门店对比"** 进入新功能页面

## 🔧 故障排除

### 问题1：TypeScript 报错 "Cannot find module"

**原因**：新创建的文件，TypeScript 编译器还没有完全索引

**解决方案**：
1. 在 VS Code 中按 `Ctrl+Shift+P`（Mac: `Cmd+Shift+P`）
2. 输入 "TypeScript: Restart TS Server"
3. 选择并执行

或者直接重启 VS Code

### 问题2：前端编译错误

**解决方案**：
```bash
cd frontend-react
rm -rf node_modules
npm install
npm run dev
```

### 问题3：后端 API 404

**检查**：
1. 确认后端已启动
2. 访问 `http://localhost:8000/api/docs` 查看 API 文档
3. 检查是否有 "全量门店对比" 相关的 API 端点

### 问题4：页面显示空数据

**原因**：数据库中没有多门店数据

**解决方案**：
1. 准备包含所有门店数据的 Excel/CSV 文件
2. 确保每行数据都有 `门店名称` 字段
3. 通过 "数据管理" 页面上传数据

## 📝 注意事项

1. **首次访问**：如果数据库中没有数据，页面会显示空状态
2. **日期范围**：与"经营总览"共享全局日期筛选器
3. **环比数据**：需要至少两周的数据才能显示环比
4. **性能**：25-30家门店的数据量，系统可以流畅运行

**解决**：重启 TypeScript Server 或重启 VS Code
