## 🆘 故障排除指南

### 启动问题
**看板启动失败** → `快速开始指南.md` - Q1
**浏览器无法访问** → 检查端口8503是否被占用

### 数据问题
**上传后没反应** → `快速开始指南.md` - Q2
**提示缺少字段** → 检查Excel中是否有"一级分类"列
**毛利率全为0** → `快速开始指南.md` - Q3

### 分析问题
**看不到分类数据** → 确认数据中有"一级分类"字段
**HHI指数不理解** → `快速开始指南.md` - Q4
**矩阵不会用** → `快速开始指南.md` - Q5

## 📊 数据准备检查表

使用前确认数据符合以下要求:

### 必需字段 ✅
- [ ] 订单ID
- [ ] 商品名称
- [ ] 商品实售价
- [ ] **一级分类**（核心字段）

### 可选字段（增强分析）
- [ ] 三级分类（用于Tab2详细分析）
- [ ] 商品成本（用于计算毛利率）
- [ ] 渠道（用于渠道分析）

### 数据质量
- [ ] 一级分类字段无空值
- [ ] 商品实售价均为数字
- [ ] 订单ID无重复（同一商品在同一订单中）
- [ ] 数据量适中（建议<100,000行）
