## ✅ **数据质量检查清单**

### **格式要求**
- [ ] 文件格式：.xlsx (推荐) 或 .csv
- [ ] 编码格式：UTF-8 (避免中文乱码)
- [ ] 日期格式：YYYY-MM-DD (如：2024-09-01)
- [ ] 数字格式：纯数字，不包含¥、￥符号
- [ ] 商品名称：去除【】()等特殊符号

### **数据质量**
- [ ] 删除空行和重复记录
- [ ] 确保必需字段无空值
- [ ] 商品名称在各Sheet中保持一致
- [ ] 价格为正数，销量为非负整数
- [ ] 时间范围尽量一致

### **隐私保护**
- [ ] 客户信息已脱敏或删除
- [ ] 具体地址可简化为区域
- [ ] 敏感供应商信息可用代码替代
